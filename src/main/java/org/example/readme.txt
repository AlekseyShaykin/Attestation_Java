Магазин игрушек.

В программе имеется один абстрактный класс, класс Toy, класс View и реализован интерфейс Actions.

Класс Toy имеет следующие свойства:
- id игрушки,
- название игрушки,
- количество игрушек на складе
- частота выпадения игрушки (вес)

При выборе пунктов меню и с помощью соответствующих методов можно:

1. Внести новую игрушку с обозначенными выше свойствами. Новая позиция автоматически будет записываться в файл note3.txt.
2. Вывести остатки игрушек на складе (читает файл note3.txt).
3. Запустить розыгрыш игрушек. Розыгрыш производится путем случйного выбора какой-либо позиции из остатков на складе.
   Выпавшая игрушка записывается в отдельный файл note4.txt, при этом количество игрушек на основном складе уменьшается.
4. Вывести список выигранных игрушек (читает файл note4.txt).
